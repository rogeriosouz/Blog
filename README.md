<h2 align="center">Clone Project</h2>

```bash
  # Clone the repository
  ❯ git clone https://github.com/rogeriosouz/blog

  # Enter directory
  ❯ cd blog
```

**Using yarn**

```bash
  # Install the dependencies
  ❯ yarn

  # Start the project
  ❯ yarn dev:server
```

**Using npm**

```bash
  # Install the dependencies
  ❯ npm install

  # Start the project
  ❯ npm start
```
